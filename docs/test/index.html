使用说明

1. 测试投票选择:

  • 点击"测试投票选择"按钮
  • 在弹出的模态框中选择任意选项
  • 确认选择并提交
  • 查看选择结果反馈

2. 测试结果查看:

  • 点击"测试结果查看"按钮
  • 在结果模态框中点击"Decrypt Results"
  • 查看模拟的投票结果和统计
  • 观察获胜选项的高亮显示

3. 真实使用:

  • 返回主页 (/) 进行真实的投票
  • 连接MetaMask钱包
  • 参与真实的FHEVM加密投票
  • 体验完整的隐私保护投票流程
